Machine Learning for Content Refresh Prioritization

Author: Asfina M

Abstract

This project investigates whether machine learning can identify webpages that require content updates. A supervised machine learning workflow was applied using the FlyRank ML Internship dataset. The trained model was compared against a baseline using the same validation strategy. The results showed improved predictive performance while maintaining honest evaluation practices. The project is intended as a decision-support tool for content teams.

Introduction

Content teams often manage thousands of webpages, making it difficult to identify which pages should be updated first. This project addresses that challenge by using machine learning to prioritize content refresh opportunities based on historical search-performance patterns.

The work was completed as part of the FlyRank Machine Learning Internship using the public-safe internship dataset. The objective is to support SEO and content teams with data-driven recommendations while ensuring that all final decisions remain under human review.

Dataset

This work uses the FlyRank ML Internship dataset. Public-safe aggregated features were used throughout the project. No client names, URLs, or private queries are included.
